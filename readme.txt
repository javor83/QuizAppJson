Приложението чете quiz.json от главната папка
За да генерираме нов файл,
1. кода в конструктора QuizSelect() се коментира
2. HomeController / Index - разкоментираме  досегашно маркираното и се връща Json
3. Клас QuizMVC , метод AddMVC - в структурата се добавя новия въпрос и наличните отговори. Оставил съм да е верен първия възможен. Но може и да се промени
4. Обратно в HomeController / Index - коментираме създаването и връщаме return View(....)
5. кода в конструктора QuizSelect() се разкоментира
